Example Mockup Home Page

Note: To use Easyrtc you must install Node.js first, download the Easyrtc GitHub repo, follow its install instructions, and then update the javascript file associated with the html live chat support files to point to your own hosted Easyrtc Node.js Server instance.

Visitors can connect, set the nickname for chat and message the admin if the admin is online, the Admin can connect and see all visitors online, get messages from all visitors into a single chat display and send messages to any visitors individually only.
